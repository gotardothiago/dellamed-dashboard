# Dellamed Dashboard

Dashboard executivo para monitoramento de vendas, expansão LATAM e performance da Dellamed.

## 🚀 Deploy no Streamlit Cloud

1. Acesse [share.streamlit.io](https://share.streamlit.io)
2. Conecte sua conta GitHub
3. Selecione este repositório
4. Clique em "Deploy"

## 📊 Funcionalidades

- **KPIs Principais**: Faturamento, pedidos, ticket médio, churn
- **Análise Regional**: Faturamento por região (Brasil, México, Colômbia, Argentina)
- **Evolução de Vendas**: Gráficos de linha temporal
- **Expansão LATAM**: Progresso de entrada em novos mercados
- **Feiras e Eventos**: Calendário de eventivos do setor
- **Alertas**: Ações prioritárias e pendências
